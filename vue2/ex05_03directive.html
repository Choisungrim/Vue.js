** Vue Template - Directives **

	=> 디렉티브는 Vue 로 화면의 요소를 더 쉽게 조작하기 위하여,
	   화면 조작에서 자주 사용되는 방식들을 모아 디렉티브 형태로 제공하는것.
	   ( v- 로 시작하고 다양한 디렉티브가 존재함. )
	=>  v-show , v-if
		- true 면 표시되고 false 면 표시되지않음 
		- 둘의 차이점은 false 의 경우 if는 삭제, show는 css 효과(display:none)
