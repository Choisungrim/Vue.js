	** Watch **
	=> 데이터 변화를 감지하여 자동으로 특정로직을 수행함, computed 와 유사하지만,
	   computed 속성은 내장 API를 활용한 간단한 연산 정도만 적합한 반면,
	   watch 속성은 데이터 호출과 같이 상대적으로 시간이 더 많이 소모되는 비동기처리에 적합.
	=> v-model        
	   양방향 데이터를 공유할때 사용
	   html 요소에서 값을 변경하면 model 객체의 값도 변경
	   텍스트박스 뿐만 아니라 다양한 입력 폼에서도 사용 가능.
