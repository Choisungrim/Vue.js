    ** Vue Template **
    => 뷰 인스턴스에서 정의한 데이터 및 로직들을 연결하여 브라우저에서 볼수있도록 HTML로 변환해주는 속성 
    => 템플릿 문법은 크게 데이터 바인딩과 디렉티브로 나뉨
    => 2가지 방법 
    	- 뷰인스턴스의 template 속성 활용방법
    	- 싱글화일 컴포넌트체계의 template 코드 활용방법
    
    ** Data Binding **
    => HTML 속성값과 뷰인스턴스 데이터를 연결
    => 주요 문법은 {{ }} 와 v-bind 속성 이 있음
