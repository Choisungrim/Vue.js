** Vue Event(Child -> Parent) Emit Sample **

    ** 이벤트로 전달 **
    => 하위(자식) -> 상위(부모) 컴포넌트 로 데이터 전달을 위해 사용됨.
    => 하위 컴포넌트에서 이벤트를 발생시켜 상위 컴포넌트에 신호를 보내면
    상위 컴포넌트에서는 이를 수신하여 메서드를 호출함.
    => 이러한 방식으로 상위 컴포넌트의 메서드를 실행할수도 있고,
    하위 컴포넌트로 내려보내는 props의 값을 조정할수도 있음.   
    
    ** 방법 **
    => 이벤트 발생
    this.$emit('이벤트명'); -> this 는 하위 컴포넌트를 가리킴
    => 이벤트 수신
    v-on:이벤트명="상위컴포넌트 메서드명"
    ( v-on -> 화면요소의 이벤트 감지 )
    
    ** 실습 **
    => child-component 의  show 버튼을 클릭하여 이벤트를 발생시키고,
    이 이벤트로 상위(Root) 컴포넌트의 printText() 메서드를 실행시킴
