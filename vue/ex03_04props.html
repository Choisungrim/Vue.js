** Vue Props Test **

{{message}} 도 가능하지만, props Test를 하기위해 componnet 호출 
   ** Vue Props 속성 **
    => 상위(부모) -> 하위(자식) 컴포넌트 로 데이터 전달을 위해 사용됨.
    => 전역 또는 지역 컴포넌트를 등록하면 등록된 컴포넌트는 하위(자식), 
       컴포넌트를 등록한 인스턴스는 상위(부모) 컴포넌트가 됨. 
    => 컴포넌트 등록과 동시에 뷰인스턴스 자체가 상위(부모) 컴포넌트가 되고
       이렇게 새컴포넌트를 등록한 인스턴스는 최상위(Root) 컴포넌트 라고함.
       항상 최상위 경로는 el 로 설정된 영역이다.

    ** 방법 **
    => 하위 컴포넌트 등록시
       props: ['props 속성 이름'] ==> 배열 Type
    => 상위 컴포넌트의 HTML 코드에 등록시
       child-component Tag 에 v-bind:propsdata="message_상위컴포넌트의 data속성" 추가함
       ( v-bind -> HTML Tag 기본속성과 Vue 데이터 속성을 연결함 )
       ( 상위의 data 값을 propsdata 속성에 전달)
